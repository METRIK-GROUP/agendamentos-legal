Política de Privacidade

App: AgendamentosRosar
Última atualização: 8 de abril de 2026

1. Dados coletados

Este aplicativo acessa dados da API do Instagram e Facebook exclusivamente para publicação e agendamento de conteúdo. Nenhum dado de terceiros é coletado, armazenado ou compartilhado.

2. Uso dos dados

Os dados acessados (tokens de autenticação, IDs de conta) são utilizados apenas para:

  • Publicar e agendar posts no Instagram
  • Publicar no Threads
  • Gerenciar conteúdo em Páginas do Facebook

3. Armazenamento

Tokens de acesso são armazenados localmente no dispositivo do usuário, nunca em servidores externos.

4. Compartilhamento

Nenhum dado é compartilhado com terceiros.

5. Exclusão de dados

Para solicitar exclusão de dados, entre em contato: rodrigo@rodrigorosar.com.br
